# 🍚 电饭煲定时计算器 (Rice Cooker Timer Calculator)

一个美观实用的网页小程序，帮助您计算电饭煲需要设置的定时时长，让您在计划的时间准时开始煮饭。

![Preview](https://img.shields.io/badge/Status-Ready-green) ![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ 功能特点

- 🕐 **实时时间显示** - 当前时间每秒自动更新
- 🎯 **滑动选择时间** - 通过滑块直观选择目标煮饭时间
- ⚡ **快捷按钮** - 一键选择常用时间（6:00、6:30、7:00、7:30、8:00）
- 📊 **智能计算** - 自动计算从现在到明天目标时间的定时时长
- 🎨 **精美界面** - 暗黑主题、玻璃态设计、流畅动画

## 🖼️ 界面预览

- 暗色渐变背景，搭配动态浮动光晕效果
- 玻璃态卡片设计，半透明模糊效果
- 可视化时间轴展示定时起止时间
- 响应式设计，适配手机和电脑

## 🚀 使用方法

### 在线访问
直接打开 `index.html` 文件即可使用

```bash
# 使用任意静态服务器，例如：
npx serve .
```

## 📝 使用说明

1. 打开应用，查看当前时间
2. 使用滑块或快捷按钮选择明天开始煮饭的时间
3. 查看计算结果，得到需要在电饭煲上设置的定时时长
4. 在电饭煲上设置对应的定时时间
